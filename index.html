What is pokey?

Pokey is an HTTP Client of sorts which allows you to specify HTTP endpoints, request headers, request body, and make assertions about the response. It can be used for regression testing, unit testing, TDD, stress testing, health checks, or even as a friendly alternative to curl.

So I downloaded it, and it looks like the whole thing is just a single Python file.

Right, the core value add is in the configuration and assertions. Pokey leverages the Requests Module under the hood, and focuses on making test creation simple. Requests is currently the only non-native module in the project, and can be installed by running setup.sh, included in the root of this project.
